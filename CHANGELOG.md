# Changelog

## [4.1.0] - 2026-01-23

### Added
- Noto Sans TC 字體支援
- 載入狀態 spinner 動畫
- Bootstrap Alert 錯誤訊息系統
- 完整 SEO metadata（Open Graph, Twitter Card）
- 時間戳記錄功能（ISO 8601 格式）
- JSON 匯出功能

### Changed
- 色彩系統簡化為白/紫/黑/黃四色
- 樣式系統採用 8px 基礎單位
- 流動排版系統（clamp() 響應式）
- 按鈕與 Alert 重新設計
- 動畫速度優化（popIn 提升 33%）
- 卡片設計簡化（移除漸層）

### Fixed
- 暗色模式文字顏色問題
- 表單輸入文字可見度
- Placeholder 提示文字顏色
- 卡片文字在暗色模式隱沒
- 名單為零時錯誤提示順序
- 參與者計數 TypeError
- Modal 未關閉問題

### Improved
- 觸控目標最小 44px
- WCAG AA 對比度標準
- 視覺噪音減少 30%
- 動畫效能優化
- 錯誤訊息友善度
